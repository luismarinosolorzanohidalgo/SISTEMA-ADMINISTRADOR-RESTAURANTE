# 🍽️ SISTEMA-RESTAURANTE

Sistema de gestión para restaurantes desarrollado en **PHP y MySQL**, diseñado para administrar de manera eficiente los pedidos, productos del menú y el control de ventas. Permite optimizar la gestión del restaurante mediante el registro de órdenes, control de productos y seguimiento de las operaciones realizadas.

---

## 🛠 Funcionalidades principales

- ✅ Gestión de productos del menú
- ✅ Registro y control de pedidos
- ✅ Administración de clientes
- ✅ Control de ventas
- ✅ Seguimiento de órdenes
- ✅ Reportes de ventas

---

## 💻 Tecnologías utilizadas

- PHP  
- MySQL  
- HTML  
- CSS  
- JavaScript
